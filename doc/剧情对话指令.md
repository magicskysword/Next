# 剧情对话指令

## 1 简介
剧情对话指令是剧情文本里使用的指令，该指令的设计是为了能更好的进行剧情编写以及在剧情文本中穿插命令。

## 2 基本格式
一条剧情指令的格式如下：

`指令名*参数一#参数二#参数三......`

而对话指令的指令名为 `Say` 与 ` `(空)
即可以省略不写，如

`Say*主角#你好。`

与

`*主角#你好。`

与

`主角#你好。`

这种写法是等价的。
